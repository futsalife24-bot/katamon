# 引継書:カタパルトモンスター(カタモン)実装

**作成:2026-07-20/仕様確定:Fable 5とのグリルセッション/実装担当:Sonnet 5**

## プロジェクト概要
縦画面スマホブラウザ向けのターン制砲撃ゲーム。モデルはポトリス/ガンバウンドM。
「片手で開いて30秒〜1分で1戦」→ 連勝を伸ばしてオンラインランキングに載せるのが核。

- 正式名:**カタパルトモンスター** 略称:**カタモン**
- 公開先:新規公開リポジトリ `katamon` → GitHub Pages(`https://futsalife24-bot.github.io/katamon/`)
- 技術:単一 `index.html`(vanilla JS + Canvas・ビルド不要)+ `assets/`(キャラ画像)+ `gas/`(ランキングAPI)

## 確定仕様(グリルで決定済み・変更時はユーザー承認必須)

### ゲームモード
1. **CPU戦の連勝サバイバル**(1人用)。負けるまで連戦、CPUが徐々に強くなる
2. スコア=連勝数を主、累計与ダメ効率を従とした複合値

### 戦場・画面
- **縦画面(9:16)・1画面完結**。スクロールなし。自分が左下・敵が右下配置
- 横幅が狭いぶん高い山なり弾道で縦空間を活かす(迫撃砲的な撃ち合い)
- **地形はランダム生成**:骨格パターン(片方が高台/中央に山/谷底スタート等)+乱数揺らし
- **地形は破壊可能**(着弾地点が削れる)。足場を失うと落下(落下ダメージあり)

### 操作
- **照準:引っ張り式**(Angry Birds式)。キャラから逆方向にドラッグ→離して発射。角度+パワー一体
- 発射前に弾道の**最初の2割だけ**点線プレビュー表示
- **移動:毎ターンあり(燃料制)**。1ターンに数歩ぶん。画面下の◀▶長押しボタン(照準操作と干渉させない)

### 対戦ルール(1試合)
- **1vs1**(プレイヤー1体 vs CPU1体)、HP制。敵HP0で勝ち
- 場外落下・自滅は即死
- **各10ターン上限**。到達時は残HP率で判定(引き分けなし、判定負け=連勝終了)
- ダメージ=直撃>至近>爆風の距離減衰+落下ダメージ
- **風:毎ターン変化**(左右方向+強さランダム)。画面上部に風向メーター

### キャラクター(3体)
- **権利方針(最重要)**:モンスターファームのキャラは使用禁止。**系統の面影だけ借りたオリジナルモンスター**(名前・デザインとも自作)。絵はAI生成(「一本足で大きな一つ目、ただし既存キャラとは別デザイン」といった指示で生成)
- 性能差(ガンバウンドMのモビル個性方式):
  - **目玉系**:標準型。素直な弾道の万能キャラ
  - **岩系**:重戦車型。HP高・爆風大、弾速遅く風に流されやすい
  - **鳥系**:トリッキー型。HP低・弾が軽く飛距離◎・移動燃料多め
- 各キャラ**必殺ショット(1試合1回)**:例=3連弾/地形大破壊/貫通弾
- CPU側も同じ3体を使用

### CPU AI(精度成長型)
- 毎回「理論上の正解弾道」を計算し、わざと誤差を乗せる。連勝が伸びるほど誤差縮小
- 1〜3連勝:誤差大/〜10連勝:誤差中/10連勝〜:誤差小+CPUも必殺使用/20連勝〜:ほぼ正解弾道
- 10連勝ごとに「ボス風の強キャラ演出」(見た目の煽りのみ、ステ盛りはしない)
- 数値はプレイテストで調整

### ランキング(GAS + スプレッドシート)
- 登録名:初回にニックネーム入力(認証なし)。端末IDをlocalStorageに保存し**自己ベスト更新時だけ上書き**
- 送信:連勝が途切れた瞬間に自動送信
- 表示:トップ20+自分の順位。**月間リセット(毎月1日)**
- 対策:GAS側で1日の送信回数上限+異常値(連勝999等)の弾きのみ。本格的な不正対策はしない(友人内スコープ)
- GASデプロイはユーザーのGoogleアカウントで行う。実装セッションで手順をガイドすること

### 演出・サウンド
- **効果音あり・BGMなし**。WebAudioで合成(音源ファイル不要):発射・爆発・落下音
- サウンドON/OFFトグル(初期値ON)
- エフェクト:爆発パーティクル/着弾時画面シェイク/ダメージ数字ポップ/必殺時カットイン風フラッシュ
- 連勝演出:勝利時「○連勝!」大表示、10連勝の節目は特別演出
